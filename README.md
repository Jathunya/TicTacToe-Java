# ❌ Tic-Tac-Toe Game ⭕

A classic **Tic-Tac-Toe** desktop application built with **Java** using **Java Swing (GUI)**. This project demonstrates core desktop application concepts, including layout management, game state control, and event handling.

## 🚀 How to Run the Application

### Prerequisites
*   Java Development Kit (JDK) 8 or higher installed on your system.

### Steps to Run
1.  Clone this repository to your local machine:
```bash
    git clone [https://github.com/your-username/tic-tac-toe-java.git](https://github.com/your-username/tic-tac-toe-java.git)
    ```
2.  Open the project in your preferred IDE (e.g., IntelliJ IDEA, Eclipse, or VS Code).
3.  Locate the file containing the `main` method and run the application to launch the game window.
